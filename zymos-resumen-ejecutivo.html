¿Qué es Zymos?

Zymos es una plataforma de gestión integral diseñada exclusivamente para cervecerías artesanales. Centraliza en un solo lugar todos los procesos del negocio: desde la producción y el inventario hasta las ventas, la facturación y el análisis financiero.

Las cervecerías artesanales enfrentan el reto de gestionar simultáneamente recetas, lotes de producción, insumos, equipos, clientes y facturación — frecuentemente con herramientas dispersas como hojas de cálculo, cuadernos y aplicaciones independientes que no se comunican entre sí. Zymos resuelve este problema ofreciendo una plataforma unificada, accesible desde cualquier navegador, sin necesidad de instalar software.

Como plataforma SaaS multi-tenant, cada cervecería opera en su propio entorno aislado con su marca, colores y datos privados — mientras comparte la misma infraestructura robusta y actualizada automáticamente.

Módulos Principales

🍻 Trazabilidad de Lotes

  • Registro completo de cada lote de cerveza
  • Seguimiento por fases: fermentación, acondicionamiento, maduración y carbonatación

  • Curva de fermentación con gráficas en tiempo real

  • Exportación a PDF por lote

  • Alertas de vencimiento próximo
  • Descuento automático al crear un lote
  • Historial completo de movimientos
  • Historial de precios por insumo y proveedor
  • Exportación a Excel

📋 Recetas

  • Biblioteca de recetas con versionado
  • Ingredientes, escalones de macerado y adiciones de hervor
  • Calculadora de escala de volumen
  • Estimación de costo por ingrediente
  • Exportación a PDF
  • Vinculación directa con lotes producidos

🔧 Equipos y Mantenimiento

  • Inventario de equipos con estado en tiempo real
  • Registro de mantenimientos preventivos y correctivos
  • Alertas de próximo mantenimiento

💰 Ventas y Clientes

  • Cotizaciones, pedidos y despachos
  • Múltiples lotes y formatos por venta
  • Remisiones generadas automáticamente
  • Ficha completa de clientes con datos fiscales
  • Listas de precio diferenciadas por canal
  • Exportación a PDF y Excel

🧾 Facturas de Proveedores

  • Registro y seguimiento de facturas de compra
  • Estados: Recibida → Verificada → Pagada
  • Asignación de costos directamente a lotes

  • Historial de cambios de estado
  • Exportación a Excel con detalle de ítems

📅 Planificación de Producción

  • Calendario visual con FullCalendar
  • Elaboraciones planificadas vinculadas a recetas

📊 Reportes y Dashboard

  • Dashboard personalizable con estadísticas en vivo
  • Reporte de producción con gráficas y métricas
  • Comparativa entre lotes (hasta 6 simultáneos)
  • Exportación a PDF y Excel de todos los reportes
  • Alertas automáticas diarias por email

Beneficios Clave

  • 🎯
    Trazabilidad completa de extremo a extremo Desde la compra de insumos hasta el despacho al cliente, cada lote tiene un historial auditado: ingredientes utilizados, costos asignados, fases de producción, evaluaciones sensoriales y ventas realizadas.
  • 💵
    Control financiero en tiempo real Costo real por litro producido, márgenes de venta, facturas pendientes de pago, ingresos despachados y top de clientes por facturación — todo en el dashboard sin exportar ni calcular manualmente.
  • ⚠️
    Alertas proactivas que evitan pérdidas Notificaciones automáticas de bajo stock, vencimientos de insumos, mantenimientos pendientes y facturas sin procesar — disponibles en la app y enviadas por email cada mañana.
  • 👥
    Roles y permisos por área de trabajo Cinco roles diferenciados (Administrador, Producción, Inventario, Facturación, Equipos) garantizan que cada colaborador acceda solo a lo que necesita, sin exponer información sensible.
  • 🌐
    Acceso desde cualquier lugar, sin instalación Plataforma 100% web. Funciona desde computadores, tablets y teléfonos. No requiere instalar ni actualizar software — los cambios y mejoras se despliegan automáticamente.
  • 🎨
    Marca propia por cervecería Cada cliente tiene su propio subdominio, paleta de colores, logo y tipografía. La plataforma se ve como si fuera desarrollada exclusivamente para su marca.
  • 📤
    Importación y exportación de datos Migración sencilla desde hojas de cálculo existentes mediante plantillas Excel prediseñadas. Exportación a PDF y Excel en todos los módulos para informes, remisiones y declaraciones.

Arquitectura SaaS Multi-Tenant

Zymos opera como una plataforma compartida donde cada cervecería es un tenant independiente con total aislamiento de datos:

🌐

Subdominio propio

Cada cliente accede por su propio dominio: micerveceria.app.com

🔒

Datos aislados

Ningún cliente puede ver ni acceder a datos de otro

🎨

Branding propio

Logo, colores y tipografías personalizadas por cliente

⚡

Actualizaciones

Mejoras desplegadas para todos los clientes simultáneamente

Esta arquitectura permite escalar el número de clientes sin incremento proporcional de costos operativos, manteniendo la seguridad y el rendimiento para todos.

Stack Tecnológico

Zymos está construido sobre tecnologías modernas, probadas en producción y con soporte de largo plazo:

Capa Tecnología Propósito
Backend Spring Boot 3.4 + Java 21 Motor principal de la aplicación — LTS hasta 2031
Base de datos PostgreSQL 16 Almacenamiento relacional con soporte multi-tenant
Frontend Thymeleaf + Bootstrap 5 Interfaz web responsiva, compatible con todos los navegadores modernos
Gráficas Chart.js + FullCalendar Visualizaciones interactivas en el dashboard y calendario
Seguridad Spring Security + JWT Autenticación, autorización por roles y API REST segura
Exportación OpenPDF + Apache POI Generación de documentos PDF y archivos Excel
Despliegue Docker + Docker Compose Contenedorización para despliegue reproducible en cualquier servidor
Migraciones BD Flyway Control de versiones del esquema de base de datos
API REST Spring MVC + OpenAPI 3 API documentada con Swagger UI para integraciones externas

Listo para producción

Zymos incluye monitoreo de salud (Actuator), métricas Prometheus, protección contra fuerza bruta,
rate limiting en la API, HTTPS headers de seguridad y perfiles de configuración para entornos de desarrollo y producción.
