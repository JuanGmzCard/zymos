Instrucciones generales
  1. Descargue la plantilla del módulo que desea migrar.
  2. Abra el archivo en Excel u OpenOffice y complete los datos desde la fila 4.
  3. Guarde el archivo sin cambiar las hojas ni los encabezados.
  4. Cargue el archivo usando el formulario correspondiente.
  5. Los campos marcados con * en la plantilla son obligatorios.
  6. Para el módulo Comercial, llene en orden: Proveedores → Facturas → Factura_Items.
  7. Para el módulo Producción, llene en orden: Recetas → Ingredientes → Escalones → Adiciones → Lotes → Lote_Ingredientes.
  8. Para el módulo Ventas, llene en orden: Ventas → Venta_Items.
  9. Para el módulo Órdenes de Compra, llene en orden: OC → OC_Items.
  10. Para el módulo Seguimiento, las 3 hojas son independientes.
